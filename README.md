# YOADAN's Coffee Hub ☕️

Welcome to **YOADAN's Coffee Hub**! This project showcases a simple landing page for a coffee shop, highlighting the products and their prices. Built with React, HTML5, and CSS3, this landing page provides a clean and inviting presentation of the coffee hub's offerings.
