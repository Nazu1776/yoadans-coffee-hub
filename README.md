YOADAN's Coffee Hub

Welcome to **YOADAN's Coffee Hub**, the ultimate destination for coffee and tea lovers! This project showcases a beautiful landing page with a selection of premium beverages, including coffee, cappuccino, tea, macchiato, and more, each available for purchase.

## Features

- A dynamic product display with images, names, and prices in Ethiopian Birr.
- Responsive design using CSS Grid to ensure an optimal viewing experience on desktops, tablets, and mobile devices.
- JavaScript used to dynamically load products and manage interactions.
- Simple and clean layout with a modern aesthetic.

Usage
The landing page will display a list of products, including coffee, tea, cappuccino, milk, macchiato, and espresso.
Each product has a name, price in Ethiopian Birr (ETB), and a button to "Add to Cart" (though no functionality has been implemented for cart management in this version).
The page is fully responsive and will adjust its layout based on the device's screen size.
Technologies Used
HTML - Used for structuring the content of the page.
CSS - Styling for layout, colors, typography, and responsive design.
JavaScript - Dynamic generation of product items and interactions.
Ethiopian Birr (ETB) - The currency used for product pricing.
