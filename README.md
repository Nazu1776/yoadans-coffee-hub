## About

YOADAN's Coffee Hub is a landing page designed to attract coffee lovers. The page highlights various coffee products, from classic Cappuccinos to premium coffee blends, allowing users to explore offerings and contact the coffee shop.

## Features

- **Responsive Design**: Adaptable to different screen sizes.
- **Product Display**: Showcases a variety of coffee products with images and prices.
- **Contact Form**: Allows users to get in touch directly.
- **Simple Navigation**: Easy access to different sections, including menu, about, and contact.

## Technologies

- **React.js**: JavaScript library for building user interfaces.
- **HTML5 & CSS3**: For structuring and styling the landing page.
